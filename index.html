¿Qué es un Algoritmo Genético?

Un algoritmo genético es un método de búsqueda y optimización inspirado en la evolución natural. Utiliza mecanismos como la selección, el cruce y la mutación para evolucionar una población de soluciones y encontrar la mejor posible en función de una función de aptitud.

¿Qué es algoritmos_geneticos_DG_PV?

Es una librería flexible que permite diseñar, ejecutar y personalizar algoritmos genéticos de forma profesional, en Español.

Selección

Ruleta, Torneo, Aleatorio.

Cruce

Un punto, Dos puntos, Uniforme.

Mutación

Básica, Aleatoria.

Reemplazo

Generacional, Elitismo.

🎯 Selección

La selección es el proceso donde se eligen los mejores individuos de la población para reproducirse. Un buen método de selección mejora la calidad de las próximas generaciones.

Ruleta

La probabilidad de ser elegido es proporcional a la aptitud del individuo.

Torneo

Se selecciona el mejor de un grupo aleatorio de individuos.

Aleatorio

Los individuos son seleccionados completamente al azar, sin considerar su aptitud.

🔀 Cruce

El cruce combina la información genética de dos padres para crear nuevos individuos (hijos). Es un mecanismo clave para explorar nuevas soluciones en el espacio de búsqueda.

Un Punto

Se elige un punto de corte y se combinan los genes de los padres a partir de ahí.

Dos Puntos

Se seleccionan dos puntos de corte, intercambiando el segmento intermedio entre los padres.

Uniforme

Cada gen se intercambia aleatoriamente entre los padres con una probabilidad fija.

🧬 Mutación

La mutación introduce cambios aleatorios en los individuos para mantener la diversidad genética y evitar que la población caiga en óptimos locales.

Básica

Invierte uno o varios genes de manera controlada con baja probabilidad.

Aleatoria

Cambia un gen a un nuevo valor completamente aleatorio.

♻️ Reemplazo

El reemplazo define cómo se forma la nueva generación a partir de los padres y los hijos. Afecta la velocidad de convergencia y la diversidad genética.

Generacional

Todos los individuos padres son reemplazados completamente por nuevos hijos.

Elitismo

Se conservan los mejores padres y se complementan con nuevos hijos para formar la siguiente generación.

🚀 Instalación y Primer Uso

Para instalar la librería algoritmos_geneticos_DG_PV una vez publicada en PyPI:

pip install algoritmos_geneticos_DG_PV

Mientras tanto, para instalar localmente:

pip install .

Y para usarlo rápidamente en Python:


from algoritmos_geneticos_DG_PV.genetico import AlgoritmoGenetico
aptitud = lambda individuo: sum(individuo)
algoritmo = AlgoritmoGenetico(
    tamaño_poblacion=20,
    num_generaciones=50,
    funcion_fitness=aptitud,
    metodo_seleccion='torneo',
    metodo_cruce='un_punto',
    metodo_mutacion='basica',
    metodo_reemplazo='elitismo',
    probabilidad_cruce=0.8,
    probabilidad_mutacion=0.05
)
algoritmo.inicializar_poblacion(tamaño_genoma=10)
algoritmo.evolucionar()
mejor, puntuacion = algoritmo.obtener_mejor_individuo()
print(f"Mejor individuo: {mejor}, Aptitud: {puntuacion}")
